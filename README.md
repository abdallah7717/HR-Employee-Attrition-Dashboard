# HR Employee Attrition Dashboard

## Project Overview
This project analyzes employee attrition using Power BI. The dashboard provides insights into employee turnover, department performance, overtime impact, job roles, gender distribution, average income, and workforce statistics.

## Objectives
- Analyze employee attrition.
- Identify departments with the highest turnover.
- Compare attrition across job roles.
- Evaluate the impact of overtime.
- Analyze average monthly income by department.
- Provide interactive HR insights using filters.

## Dashboard KPIs
- Total Employees
- Employees Left
- Attrition Rate
- Average Age
- Average Monthly Income

## Dashboard Features
- Attrition by Department
- Attrition by Job Role
- Attrition by Gender
- Attrition by Overtime
- Average Income by Department
- Interactive slicers for Department, Job Role, and Gender

## Tools Used
- Power BI
- Microsoft Excel

## Key Insights
- Total Employees: 1,470
- Employees Left: 237
- Attrition Rate: 16.12%
- Research & Development recorded the highest employee attrition.
- Employees working overtime experienced higher attrition than employees without overtime.
